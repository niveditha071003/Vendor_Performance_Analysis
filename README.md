# Vendor Performance Analysis 📊

## Project Overview

The **Vendor Performance Analysis** project focuses on evaluating vendor efficiency and procurement performance using data analytics techniques. The goal is to transform raw purchase, sales, and inventory data into meaningful insights that help businesses make **data-driven procurement decisions** and optimize vendor relationships.

This project performs **data cleaning, exploratory data analysis (EDA), and dashboard visualization** to identify top-performing vendors, analyze purchasing patterns, and detect opportunities for cost optimization. Vendor performance metrics such as purchase contribution, sales trends, and inventory movement are analyzed to support strategic decision-making. ([GitHub][1])

---

# Objectives

* Analyze vendor purchase and sales data to evaluate vendor performance
* Identify **top-performing and underperforming vendors**
* Detect purchasing patterns and inventory trends
* Generate insights that help improve procurement efficiency
* Build an **interactive Power BI dashboard** for business stakeholders
